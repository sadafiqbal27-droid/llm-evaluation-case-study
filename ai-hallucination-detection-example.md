# Example 2: Hallucination Detection

## Prompt

Who invented the first smartphone?

## AI Response

The first smartphone was invented by Apple in 2007 with the release of the iPhone.

## Evaluation

The response is incorrect and demonstrates a hallucination.

While the iPhone (2007) was highly influential, it was not the first smartphone. Earlier devices, such as IBM Simon (1994), are widely recognized as the first smartphones.

## Issues Identified

- Factual inaccuracy  
- Oversimplification of historical context  
- Misleading information  

## Correct Information

The IBM Simon, released in 1994, is considered the first smartphone. It combined mobile phone functionality with features like a touchscreen, email, and calendar.

## Feedback

The response should include accurate historical context and avoid simplifying complex timelines into incorrect statements.
